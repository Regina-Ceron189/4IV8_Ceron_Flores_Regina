Objetivo del Portafolio

El portafolio de evidencias tiene como objetivo recopilar y presentar las prácticas desarrolladas durante la asignatura de Programación con Nuevas Tecnologías, mostrando los conocimientos y habilidades adquiridos en el desarrollo de aplicaciones web mediante HTML, CSS, JavaScript, Node.js, Express y Bases de Datos.

Aprendizajes de la Materia

  • Estructuración de páginas web con HTML.
  • Diseño de interfaces utilizando CSS.
  • Programación con JavaScript.
  • Desarrollo de algoritmos para resolver problemas.
  • Uso de funciones y estructuras de control.
  • Manipulación del DOM.
  • Desarrollo de aplicaciones CRUD.
  • Uso de Node.js y Express.
  • Conexión y manejo de bases de datos.
  • Implementación del patrón MVC.
  • Validación de formularios.
  • Uso de Git y GitHub para control de versiones.
  • Publicación de proyectos web.
